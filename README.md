# Portal D&V

## Visão geral

O Portal D&V é um ecossistema digital desenvolvido para reunir, em uma única arquitetura de marca, diferentes frentes de negócio e conteúdo:

- **Seta**: palestras, treinamentos e autoridade  
- **Book Music**: composições, licenciamento musical e livros autorais  
- **Commerce & MKT**: branding, jingles, campanhas e produtos  
- **Rádios**: Rádio Memories e Rádio Maná  

A proposta do projeto é organizar essas verticais em um hub moderno, escalável e visualmente coerente, preservando a identidade própria de cada unidade.

---

## Stack utilizada

- **Next.js 14**
- **React**
- **TypeScript**
- **Tailwind CSS**
